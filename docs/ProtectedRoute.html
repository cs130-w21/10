Class: ProtectedRoute

ProtectedRoute()

Routing proxy for access protection based on login and completed profile

Constructor

new ProtectedRoute()

Checks if user is logged in, if not redirects to login page. If user is logged in and is trying to access a page that requires a completed profile, checks if they have a completed profile; if not, redirects to complete-your-profile page, otherwise directs to desired page. If user is logged in and does not need complete profile to access page, directs user to desired page.
