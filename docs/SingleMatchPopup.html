Class: SingleMatchPopup

SingleMatchPopup(isOpen, onDismiss, userProfile)

Component to render a Popup for a user's single match object

Constructor

new SingleMatchPopup(isOpen, onDismiss, userProfile)

Parameters:
Name Type Description
isOpen boolean

A boolean to control whether the popup should be displayed or hidden.
onDismiss callback

A callback that will be called when a user tries to exit the Dialog/Popup.
userProfile SingleMatchUserProfile

A JSON object that contains all the data needed to render that user's single match.

// In this example, popup stays open and has no way to be closed
