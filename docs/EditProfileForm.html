Class: EditProfileForm

EditProfileForm(onSuccessRedirectURL, onCancel)

Component to render a form allowing users to edit their profile details. Currently onSuccessRedirectURL is used by CompleteYourProfile page whereas onCancel is used by EditProfilePopup.
